# My Dotfiles

A personal dotfile manager repository using Chezmoi.

## Setup Instructions

1. Install Chezmoi:
```bash
sh -c "$(curl -fsLS get.chezmoi.io)"
```

2. Initialize with this repository:
```bash
chezmoi init git@github.com:BeechcraftNV/dotfiles.git
```

3. Apply the dotfiles:
```bash
chezmoi apply
```
